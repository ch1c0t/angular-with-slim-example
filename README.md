angular-with-slim-example
===================

Example app with sinatra, angular, slim, and livescript.

## Prerequisites

1. Ruby
2. NodeJS for Bower

## How to start?

1. `git clone https://github.com/ch1c0t/angular-with-slim-example`
2. `bundle install`
3. `bower install`
4. `bundle exec rackup -p 1111`
